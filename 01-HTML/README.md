## Elements
Most elements have opening and closing tags. Usually a tag has the following structure:

* Opening tag: `<element>`
* Closing tag: `</element>`

### HTML Elements
####Headings

* Main heading (h1) tag: `<h1>` and `</h1>`
* Subheading (h2) tag: `<h2>` and `</h2>`
* Other heading levels: h3, h4, h5 and h6

####Paragraph

* Paragraph tag: `<p>` and `</p>`

####Comment
* Opening tag: `<!--`
* Closing tag: `-->`
